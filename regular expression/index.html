Regular Expression

    General Part

  • start with /^ end with $/
  • () === optional
  • \/ === '/'
  • | === or
  • []=== include
    eg. [0-9a-z_]
  • [^] ===Negate / not include
  • {}=== the number include
    eg.\d{3} 3 digitals
  • /i === ignore case
  • /g === looping 循环匹配直到匹配不到

    • Digital Part

  • \d ===[0-9]
  • \d{2} === 2 digitals
  • \d{2,4}=== 2-4 digitals
  • \d{2,} === more than 2 digitals
  • \d+ === \d{1,} at least one digital;最少1
  • \d* === \d{0,};可有可无
  • \d? === \d{0,1} the most is one digital;最多1
  • \D ===[^0-9] not a digital

    • String Part

  • \s === [ \f\n\r\t\v] space or enter 空白和换行
  • \S === [^ \f\n\r\t\v]not a space 非空白
  • \w === [A-Za-z0-9_]
  • \W === [^A-Za-z0-9_]
  • . === [^\n\r] 匹配除换行符\n、\r之外的任何单个字符
  • \n === new line 换行
  • \r === enter 回车

    • Example Part

  • /^[\u4e00-\u9fa5]$/ 中文only
  • .*? === non greedy match
    eg.path.match(/id=(.*?)(&|$))会一直找直到找到条件2中的&或者$
  • str.replace(/<[^>]+>/g, '') === ignore html tag <..>
  • /<\/?([a-z]+).*?>/ig ===html tag included end tag

    • Function Part

  • .replace() return string
    eg. str.replace(/<\/?([a-z]+).*?>/ig, (result,group1)=>{
    if(group1==='img'){
    return result }
    return '' })
  • .test() return boolean
  • .match() return arr[result, (1)result,(2)result...(n)result]
  • .toLocaleString() digital add thousandths
